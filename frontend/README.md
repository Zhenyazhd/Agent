# LLM Agent Frontend

Frontend for interacting with the LLM Agent server. 

## Quick Start

### Prerequisites

- Node.js 22+
- Backend server running (see main [README](../README.md))

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173` (default Vite port).

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
